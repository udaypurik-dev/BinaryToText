# Binary Translator 🔄

A sleek, macOS-inspired desktop utility built with **Electron** that instantly translates English text to binary code, and vice versa. 

Designed with a stunning "liquid glass" (glassmorphism) user interface, it runs quietly in the background and can be summoned instantly from anywhere on your PC using a global hotkey!

## Features ✨

- **Two-Way Translation**: Type English to get Binary instantly, or paste Binary to decode it back into English.
- **Global Hotkey**: Press `Ctrl + Shift + P` from anywhere on your computer to instantly pop open or hide the app.
- **Liquid Glass UI**: Beautiful dark-mode glassmorphism aesthetics with macOS-style window controls.
- **One-Click Copy**: Quickly copy the translated binary to your clipboard.

## Prerequisites 🛠️

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

## Installation 🚀

1. Clone or download this repository to your local machine.
2. Open a terminal or command prompt and navigate to the project directory:
   ```cmd
   cd path/to/binary-translator
   ```
3. Install the required dependencies:
   ```cmd
   npm install
   ```

## Usage 💻

1. Start the application:
   ```cmd
   npm start
   ```
2. The translator window will appear. You can interact with it immediately.
3. Click away or minimize the app, and it will stay active in the background.
4. Hit the global hotkey `Ctrl + Shift + P` at any time to bring the window back to the front!

## Technologies Used 🧰
- **Electron** (Desktop App Framework)
- **HTML5 & CSS3** (Vanilla CSS for Glassmorphism styling)
- **JavaScript** (Translation logic & IPC)
